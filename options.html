Options for Spotify Web Extension

This extension can help your browser open "spotify:" links (see linking to spotify), but requires you grant extra permissions.
We can also add a widget on youtube (to add to your playlist) (not yet implemented, only a permission grant)
Want to skip tracks with a keyboard shortcut? Try our Skip Track extension.
Remote access: setup a remote access channel to this computer. This will require you to sign in with google, if you're not already signed in.
